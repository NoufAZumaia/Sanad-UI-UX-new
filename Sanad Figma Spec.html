كيفية الاستيراد إلى Figma
  1. افتح Figma → File → Import → اختر هذا الملف بصيغة PDF (انقر "طباعة / PDF" أعلاه واحفظ).
  2. أو استخدم إضافة html.to.design داخل Figma وألصق رابط أي من ملفات HTML.
  3. أنشئ مكتبة Variables باستخدام جدول الـ tokens أدناه (Local variables → New collection).
  4. ابدأ بنسخ شاشات الـ Frames المرجعية في الأسفل كنقطة انطلاق.
